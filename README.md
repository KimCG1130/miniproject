### 프로젝트 소개

여행하면서 찍은 사진, 느낀 점들을 공유해보아요!

### 프로젝트 개요

 여행 사진을 올리고 관련내용을 적어 게시판에 올릴 수 있습니다.

 게시글은 모두가 볼 수 있고, 댓글기능과 좋아요 기능은 로그인 한 사람만 이용할 수 있습니다.

 좋아요를 많이 받은 글이 먼저 정렬됩니다.
